# 安装&配置

## WSL上安装

### 安装linux

1. 安装
```shell
wsl --install -d <发行版名>
```

2. 进入
```shell
# 默认的发行版
wsl
# 进入指定的发行版
wsl -d Ubuntu-22.04
```
### 安装openclaw
3. 安装nodejs
```shell
sudo apt update
sudo apt install nodejs
```

4. 全局安装openclaw  
```shell
npm install -g openclaw@latest
```

5. 初始化配置
```shell
openclaw init
```
6. 启动服务
```shell
openclaw gateway start
```

7. 打开Dashboard
```shell
openclaw dashboard
```



## 在windows系统上安装

1. 检查是否安装了node.js
```shell
node -v
```

2. 安装openclaw
```shell
npm install -g openclaw@latest
```

3. 配置根目录

"我的电脑"右键"属性"->"高级系统设置"->"环境变量"->"系统变量"

添加一个环境变量，变量名设置为OPENCLAW_CLAW，变量值设置为你想设置的目录。保存。

重新打开PowerShell，输入```echo $env:OPENCLAW_CLAW```，查看是否配置成功。

4. 初始化配置
```shell
openclaw onboard
```

5. 配置openclaw开机自启




## 基础配置

| 配置文件 | 描述 |
| --- | --- |
| BOOTSTRAP.md | openclaw初始化时的工作 |
| IDENTITY.md | openclaw的名字、性格、身份、角色等 |
| SOUL.md | openclaw的个人特征、情感、行为等 |
| USER.md | 用户的名字、身份、性格、国家等 |

配置完重启下openclaw：
```shell
openclaw gateway restart
```
