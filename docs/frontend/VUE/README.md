### var、let、const的区别

var用于声明变量，且具有”变量提升“的特性，因而不推荐使用。

> 什么是变量提升？

let、const是ES6后新增的关键字。

let负责声明变量；const负责声明常量。  

const声明的常量只可读，不可被修改。定义时必须初始化。
