






## 快捷键

必须要会

| 功能描述             | SAI            | Krita     |
|------------------|----------------|-----------|
| 矩形选区             |                | Ctrl + R  |
| 自由变换，按Enter结束    | 	Ctrl + T      | Ctrl + T  |
| 撤销	              | Ctrl + Z       | Ctrl + Z  |
| 画直线	             | 按住 Shift 点击或拖动 |


工具切换

| 功能描述  | SAI  | Krita |
|-------|------|-------|
| 铅笔工具  | 	N   | B     |
| 喷枪工具  | 	B   |
| 笔工具   | 	V   |
| 水彩笔工具 | 	C   |
| 橡皮擦工具 | 	E   | E     |
| 选区笔工具 | 	A   |
| 选区擦工具 | 	S   |

画笔控制

| 功能描述           | SAI     | Krita |
|----------------|---------|-------|
| 增大笔刷大小         | 	]      |
| 减小笔刷大小         | 	[      |
| 调整笔刷浓度（0为100%） | 	0-9    |
| 切换前景色/背景色      | 	X      | X     |
| 取色             | ALT+左键  | 右键    |


视图操作

| 功能描述          | SAI          | Krita      |
|---------------|--------------|------------|
| 抓手工具（随意移动画布）  | 	空格键 + 左键拖动  | 空格键 + 左键拖动 |
| 左右翻转画布（检查造型）  | 	H           | M          |


图层操作

| 功能描述    | SAI            | Krita   |
|---------|----------------|---------|
| 新建图层    | Insert（需要自己设置） | Insert  |
| 与下一图层合并 | 	CTRL+E        | CTRL+E  |
| 清空当前图层  | Delete         | Delete  |
