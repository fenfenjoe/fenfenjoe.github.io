# SAI笔记

- 如何查看画笔、工具的快捷键？
  - 方法1：对画笔鼠标右键->属性，便可看到快捷键
  - 方法2：工具栏->其他->快捷键设置 查找

- 怎么样“框选内容-移动”？
  - 选择框选工具（P）-> 框选内容 -> 按住Ctrl -> 鼠标左键拖动 -> 回车 -> Ctrl + D
- 怎么样“框选内容-旋转”？
  - 选择框选工具（P）-> 框选内容 -> Ctrl+T -> 旋转 -> 回车 -> Ctrl + D
- 怎么样“框选内容-删除”？
  - 选择框选工具（P）-> 框选内容 -> DELETE
- 怎么样“取消选中”？
  - Ctrl + D

- 怎么样画直线
  - 方法1：选择任意画笔 -> 在起点点击鼠标左键 -> 按住Shift -> 在终点点击鼠标左键
  - 方法2：Ctrl + R（显示尺子） -> 按住Ctrl可以拖动尺子 -> 再此按Ctrl + R（隐藏尺子）
- 怎么样画空心圆
  - 尺子处，选择同心圆 -> 同样按住Ctrl可以拖动尺子 -> 再此按Ctrl + R（隐藏尺子）
