## 概览
线性代数研究什么？   
研究**行列式**、**矩阵**、**向量**、**线性方程**；


线性代数的应用？
* 解线性方程（若以这个为目标去阅读会比较方便！不然总觉得线性代数没有应用的地方）
* 机器学习
