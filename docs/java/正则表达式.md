# 正则表达式

正则表达式在线验证：https://c.runoob.com/front-end/854/

## 规则

| 正则表达式 | 描述                         | 匹配成功的字符串示例 | 匹配成功结果 | 匹配失败的字符串示例 | 匹配失败结果 |
| ---       | ---                          | ---                |---          |---                 |---          |
|^ba        |字符串以ba开头，则返回ba       |back big beauty|ba|buck back|(空)|
|ld$        |字符串以ld结尾，则返回ld       |back big build|ld|build buck|(空)|
|b.d        |匹配单个字符                   |bid bad b d bak|"bid" "bad" "b d"(三个结果)|bacd|(空)|
|b..d       |匹配两个字符                   |bild b ld b3ld|"bild" "b ld" b3ld"(三个结果)|babcd|(空)|
|b.*d       |匹配0个或多个字符               |bild b ld b3ld|"bild b ld b3ld"(一个结果，只会匹配到最长的)|babcc|(空)|
|\bb\w*d\b  |匹配0个或多个字符的单词         |bild b ld b3ld|"bild" b3ld"(两个结果)|||
|b.+d       |匹配1个或多个字符              |bccd|bccd|bd|(空)|
|b.?d       |匹配0个或1个字符              |bd|bd|b12d|(空)|
|b.{5}d     |匹配5个字符                    |b12345d|b12345d|b1234d|(空)|
|b.{5,}d    |匹配至少5个字符                |b12345d|b12345d|b12345d b123d|"b12345d b123d"(返回最长的匹配结果)|
|b.{5,7}d   |匹配5~7个字符                  |b123456d b123455d|"b123456d" "b123455d"(两个结果)|b12345678d|(空)|
|b[123]d    |匹配[]中任一个字符              |b1d b2d b23d|"b1d" "b2d"(两个结果)|b4d b23d|(空)|
|b[123]{3}d |匹配[]中任意3个字符             |b111d b132d|"b111d" "b123d"(两个结果)|b12d|(空)|
|b[^123]d   |匹配不在[]中的任一个字符         |b4d|b4d|b3d|(空)|
|b\dd       |匹配1个数字                     |b1d|b1d|bad|(空)|
|b\wd       |匹配1个数字或者字符（非空格）     |bad b1d|"bad" "b1d"(两个结果)|b d|(空)|


## 常用正则表达式

```
#匹配所有中文
[\u4e00-\u9fa5]

#所有大写字母
[A-Z]

#所有字母和数字
[A-Za-z0-9]


```
