# 英语学习笔记



## 日常
- **estimate** vt. 预估、预计 n. 估算值
  - estimate...to be...(预估xx的数量约为...)
  - It's estimated that...(预计...)

- **estimated** adj. 预估的、预计的
  - estimated time(预估时间)、estimated cost(预估成本)

- **workload** n. 工作量

- **deliver** vt. 交付

- **requirement** n. 需求

- 我觉得有可能是因为...
  - **It might be...**
  - **I suspect that...**
  - **The reason may be that**


## Git相关

git文档（可以学习相关术语）：<https://git-scm.com/>

常用短语：
- **拉取最新代码到本地**： Pull the latest code to local
- **创建一个新分支去完成功能**： Create a new branch from origin/master
- **提交代码**：Commit the changes
- **发起Pull Request(PR)并等待代码审查(CR)**：Create a Pull Request (PR) and wait for Code Review (CR).
- **撤销提交**：Revert the commits

## 介词的用法

| 介词          | 主要用法                   | 常用场景 / 例句                                             |
|-------------|------------------------|-------------------------------------------------------|
| **in**      | 时间：较长时段（月、年、世纪、一天中的部分） | in May / in 2025 / in the morning                     |
|             | 地点：在内部 / 大地点           | in the room / in Beijing / in the water               |
|             | 状态 / 方式                | in trouble / in a hurry / in English / in a red skirt |
| **at**      | 时间：具体时刻 / 短时段          | at 5 o’clock / at noon / at night                     |
|             | 地点：具体点 / 小地点           | at the door / at the station / at home                |
|             | 方向 / 目标                | look at / arrive at / aim at                          |
| **on**      | 时间：具体某天 / 带day的时段      | on Monday / on June 1st / on the weekend              |
|             | 地点：在表面上 / 附着           | on the table / on the wall / on the bus               |
|             | 关于 / 状态                | on the phone / on fire / a book on history            |
| **for**     | 目的 / 为了                | buy for you / wait for / ask for                      |
|             | 时间段（持续）                | for three days / for a long time                      |
|             | 原因 / 支持                | thank for / vote for                                  |
| **to**      | 方向 / 到达                | go to school / send to / from…to…                     |
|             | 比较 / 对应                | compare to / similar to                               |
|             | 目的（不定式）                | I came to help.                                       |
| **by**      | 方式 / 手段                | by bus / by hand / by email                           |
|             | 时间截止                   | by 5 pm / by tomorrow                                 |

## 连接副词

| 词/短语              | 含义/作用       | 例句                                                                                         |
|-------------------|-------------|--------------------------------------------------------------------------------------------|
| **因果关系**          |             |                                                                                            |
| therefore         | 因此          | The component is decoupled; **therefore**, it is easier to test.                           |
| consequently      | 因此，所以       | The service failed; **consequently**, the whole system crashed.                            |
| as a result       | 结果          | He forgot to close the connection; **as a result**, a memory leak occurred.                |
| thus              | 因此，从而       | The code is refactored, **thus** reducing technical debt.                                  |
| hence             | 因此，由此       | The API is deprecated; **hence**, we should migrate to the new one.                        |
| **递进与补充**         |             |                                                                                            |
| furthermore       | 此外，而且       | The solution is scalable; **furthermore**, it is secure.                                   |
| moreover          | 此外，更重要的是    | The design is clean; **moreover**, it meets all requirements.                              |
| in addition       | 另外          | **In addition**, we need to update the documentation.                                      |
| additionally      | 此外          | **Additionally**, the response time improved by 20%.                                       |
| besides           | 此外，而且       | The approach is simple; **besides**, it requires no extra dependencies.                    |
| **转折与对比**         |             |                                                                                            |
| however           | 然而          | The system is fast; **however**, it consumes too much memory.                              |
| nevertheless      | 尽管如此        | The task was challenging; **nevertheless**, we completed it on time.                       |
| nonetheless       | 尽管如此        | It's an old library; **nonetheless**, it's still widely used.                              |
| on the other hand | 另一方面        | This solution is cheap; **on the other hand**, it lacks flexibility.                       |
| in contrast       | 相比之下        | Java uses static typing; **in contrast**, Python is dynamically typed.                     |
| conversely        | 相反地         | Adding more threads may improve performance; **conversely**, it could increase contention. |
| **顺序与时间**         |             |                                                                                            |
| first / firstly   | 首先          | **First**, we need to define the interface.                                                |
| second / secondly | 其次          | **Secondly**, implement the business logic.                                                |
| finally           | 最后          | **Finally**, deploy the application.                                                       |
| meanwhile         | 与此同时        | The frontend team builds the UI; **meanwhile**, the backend team sets up the API.          |
| subsequently      | 随后          | The bug was fixed; **subsequently**, the build passed.                                     |
| **举例与说明**         |             |                                                                                            |
| for example       | 例如          | Many frameworks support DI; **for example**, Spring and Guice.                             |
| for instance      | 例如          | Consider a microservice architecture; **for instance**, order service and payment service. |
| such as           | 比如 (通常后接名词) | We use various tools **such as** Maven, Git, and Jenkins.                                  |
| namely            | 即，也就是       | There is one key benefit, **namely**, reduced coupling.                                    |
| **总结与结论**         |             |                                                                                            |
| in conclusion     | 总之          | **In conclusion**, dependency injection simplifies testing.                                |
| to sum up         | 总结          | **To sum up**, IOC containers manage object lifecycles.                                    |
| overall           | 总体上         | **Overall**, the migration was successful.                                                 |
| in summary        | 概括来说        | **In summary**, the new architecture improves maintainability.                             |
| **强调与重述**         |             |                                                                                            |
| in other words    | 换句话说        | The bean is singleton; **in other words**, only one instance exists.                       |
| that is           | 也就是说        | The method is idempotent; **that is**, calling it multiple times has the same effect.      |
| indeed            | 确实          | **Indeed**, the performance test confirmed our hypothesis.                                 |
| **条件与让步**         |             |                                                                                            |
| otherwise         | 否则          | We must handle the exception; **otherwise**, the application may crash.                    |
| in that case      | 那样的话        | If the database is down, **in that case**, we fall back to cache.                          |
| **类比与比较**         |             |                                                                                            |
| similarly         | 同样地         | Microservices scale independently; **similarly**, each can be deployed separately.         |
| likewise          | 同样          | The service requires authentication; **likewise**, all endpoints should be secured.        |
| **表达目的**          |             |                                                                                            |
| to this end       | 为此          | We need better monitoring; **to this end**, we will implement Prometheus.                  |
| for this purpose  | 为此          | A dedicated thread pool was created; **for this purpose**, we used Executors.              |
