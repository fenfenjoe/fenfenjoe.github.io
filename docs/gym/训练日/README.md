# 上肢训练日

## 涉及的肌群

| 部位 | 肌肉               |
|---|------------------|
|背部| 菱形肌、背阔肌、大小圆肌、冈下肌 |
|胸部| 前锯肌、胸大肌          |
|手臂| 肱二头肌、肱三头肌        |
|肩部| 三角肌、斜方肌、肩胛提肌     |


## 热身

| 热身动作         | 目的     | 注意事项 |
|--------------|--------|------|
| **胸椎屈伸**     | 胸椎灵活度  | 2*10 |
| **单腿跪姿胸椎旋转** | 胸椎旋转能力 | 双手抱头，左右旋转，2*10 |
| **龙门架肩外旋**   |激活肩袖肌群| 想象胳肢窝夹着瓶水，2*15 |
| **前平举+侧平举**  |激活肩袖肌群| 2*10 |
| **俯身飞鸟**     |激活**上背部**（三角肌后束、冈下肌、菱形肌）|10*2|
|**颈前颈后弹力带环绕**|激活肩袖肌群+背部|10*2|
