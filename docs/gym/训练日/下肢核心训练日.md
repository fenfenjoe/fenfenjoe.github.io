# 下肢训练日

## 涉及的肌群

| 部位  | 肌肉                                 |
|-----|------------------------------------|
| 核心  | 腹直肌、腹外斜肌                           |
| 臀部  | 臀大肌、臀中肌、臀小肌、梨状肌                    |
| 大腿  | 股二头肌、股四头肌                          |
| 小腿  | 胫骨前/后肌（内侧）、腓骨长/短肌（外侧）、跟腱（腓肠肌、比目鱼肌） |
| 脚踝  | 足底方肌、蚓状肌                           |



## 热身

| 热身动作      | 目的             | 注意事项 |
|-----------|----------------|------|
| **握脚尖抬臀** | 拉伸股四头肌、股二头肌、臀肌 | 2*10 |
| **2.5kg哑铃深蹲** | 拉伸股四头肌、股二头肌、臀肌 | 2*10 |
