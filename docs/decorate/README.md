### 选择全包你应该注意

注意装修套餐中包含哪些内容，了解有哪些是套餐不包含，但是自己装修需要的。  
以下一般是增项内容，即套餐一般不包含的：  
* 拆墙砌墙
* 建筑垃圾清运
* 除厨房、卫生间外的瓷砖
* 封窗
* 吊顶
* 使用更好的材料（瓷砖、柜体木材、五金配件、水电线、开关等）
* 踢脚线

## 装修流程

* 墙体拆除
* 成品保护（地面、墙体贴保护膜）
* 水电改造
  * 开槽
  * 走管
  * 电线保护（缠胶布）
  * 水管打压测试
* 卫生间
  * 回填材料
  * 下水管道包隔音材料、用龙骨或轻体砖封起来
  * 封水泥
  * 刷防水涂料
  * 闭水试验
  * 检查排水坡度
  * 地漏
  * 铺砖
* 木工
  * 电视背景墙
  * 吊顶
    * 铺龙骨
    * 封石膏板
    * 刷漆
* 瓦工
  * 贴砖
* 油工
  * 刷漆
    * 刮腻子（为了找平）
    * 刷漆
* 安装阶段
  * 厨卫吊顶
  * 橱柜、油烟机等
  * 窗帘杆、晾衣架
  * 开关
  * 灯具
  * 定制柜
  * 马桶
  * 安装完补漆
  * 开荒保洁
* 软装阶段
* 准备入住阶段
  * 去甲醛（3个月）
  * 开通电视、网络
  * 开通燃气
