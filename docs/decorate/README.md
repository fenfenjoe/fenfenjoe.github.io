# 装修那些事儿

## 装修方案阶段

不同类型的装修公司，基本可以分为：**清包**，**半包**，**全包**

| 环节   | 清包  | 半包   | 全包  |
|------|-----|------|-----|

| 辅材   | ×   | √    | √   |
| 设计   | ×   | 可选   | √   |

| 家具电器 | ×   | ×    | 可选  |
| 灯具   | ×   | ×    | 可选  |


### 选择全包你应该注意

注意装修套餐中包含哪些内容，了解有哪些是套餐不包含，但是自己装修需要的。  
以下一般是增项内容，即套餐一般不包含的：  
* 拆墙砌墙
* 建筑垃圾清运
* 除厨房、卫生间外的瓷砖
* 封窗
* 吊顶
* 使用更好的材料（瓷砖、柜体木材、五金配件、水电线、开关等）
* 踢脚线


### 选择半包你应该注意

1. 第一步最好是找到一个好的设计师，帮你设计全屋的结构、风格，出一个符合你需求的平面图、VR效果图。这样你去跟半包公司沟通、或者买材料时方便很多。
2. 半包需要自己去买主材，包括瓷砖、门、窗、橱柜、衣柜、卫浴、灯具等等等等，要有经常跑家装市场、选材料、对比价格的心理准备以及时间准备。


### 选择清包你应该注意

1. 最好是有一定的装修行业经验、设计经验的人，才选择清包。
2. 清包除了设计、主材需要自己负责，还需要自己去采购辅材。

### 板材选择攻略

#### 了解板材

| 名称       | 实木or人造板材 | 制作过程                                     | 优点                             | 缺点                    |
|----------|----------|------------------------------------------|--------------------------------|-----------------------|
| 大芯板      | 实木       | 切成一个一个木块，用胶水粘合；最后两面加表板                   | 易加工、最环保、价格低                    |                       |
| 指接板      | 实木       | 大芯板的2.0，不用胶水粘合，改用榫；最后不加表板                |                                |                       |
| 胶合板（多层板） | 实木       | 将树旋切成片，将每片叠在一起并用胶水粘合而成                   | 防潮性好                           | 易变性                   |
| 生态板      | 实木       | 在大芯板或指接板的基础上，贴一层三聚氰胺贴皮                   |                                |                       |
| 密度纤维板    | 人造板材     | 原木加工剩下的边脚料，做成刨花，加胶水搅匀，最后压成板              | 容易做造型                          | 防水性差、受潮易膨胀；承重能力差；环保性差 |
| 刨花板      | 人造板材     | 与密度纤维板差不多，不过是用刨花+其他杂料（草杆稻壳等）磨成粉做成；比纤维板便宜 | 易加工、环保性、降噪效果好                  |                       |
| 颗粒板      | 人造板材     | 不掺杂料的刨花板                                 | 比刨花板便宜、密度比刨花板更高更坚固、表面平整易上漆     | 防潮性一般                 |
| 欧松板（OSB） | 人造板材     | 也是刨花板的一种，用的纯木头片，还细分为大刨片，小刨片              | 防潮性更好，密度比颗粒板更高更坚固、表面平整易上漆、环保性好 | 比颗粒板贵                 |
| 禾香板      | 人造板材     | 也是刨花板的一种，里面掺杂了20%的秸秆碎料，另外80%是刨花          | 环保性最好                          |                       | 


#### 板材常用于的家具

橱柜、衣柜、书架、桌子、床等。


厨卫：欧松板、胶合板
衣柜：颗粒板、禾香板

#### 板材的参数

1. **材质**：板材的原材料种类和质量，如实木、人造板、胶合板等。
2. **密度**：板材的密度直接影响其强度和稳定性。
3. **厚度**：板材的厚度决定了其承重能力和使用寿命。
4. **平整度**：板材表面是否平整，有无凹凸不平影响家具的外观和质感。
5. **防潮性**：板材的防潮性能，影响其在不同环境下的稳定性。
6. **环保等级**：板材是否符合环保标准，对家具制作过程中的健康和环保都很重要。
7. **含胶量**


#### 板材的等级

国标GB/T18259规定，根据甲醛释放限量，将室内用人造板材分为3个等级：
* ENF：甲醛<=0.025mg/L
* E0：甲醛<=0.05mg/L
* E1：甲醛<=1.24mg/L

满足以上等级的，才可被称作**环保板材**


#### 常见实木

硬木：橡木、桃心木、桦木、红橡、硬枫、赤杨、榉木和黄杨
软木：针叶树、橡树树皮制作而成

#### 关于胶水

* 脲醛胶：尿素+甲醛+水，上限E0级，无下限
* MDI胶：化学胶水，ENF级
* 生物胶：ENF级，价格较贵

### 门窗选择攻略

#### 如何选择门窗？

* 选超白玻璃或双层中空玻璃
* 玻璃要带3C认证
* 封阳台窗框选**断桥铝**
* 铝材厚度选1.8mm以上
* 型材厚度（即窗框宽度），一般75以上即可，高层可选80的
* 选封窗的商家要提早，钢化玻璃定做一般要15天以上

#### 选瓷砖需要注意的问题

* 如果选有花纹的抛釉砖、木纹砖这一些，缝隙可能会很明显（因为美缝剂没有花纹）
* 一般瓷砖的光泽度越低，防污能力越难，但防滑能力会比较好

#### 瓷砖工艺需要注意

* 瓷砖一定要刷背胶
* 地砖通铺要对缝
* 地砖与墙砖也要尽量对缝
* 瓷砖阳角要做海棠角
* 卫生间瓷砖要墙砖压地砖，反着来的话容易渗水


#### 陶瓷砖类型

| 名称   | 描述                                         | 适用场景      | 
|------|--------------------------------------------|-----------|
| 陶质砖  | 陶土1000°以下高温烧制而成，不防滑                        | 基本用于墙砖    | 
| 通体砖  | 又叫无釉瓷砖，是一种胚体和砖表面颜色保持一致的低吸水率瓷砖。防滑耐磨，但可选花色不多 |           |
| 抛光砖  | 对通体砖抛光而成，表面没花纹图案，防滑、耐磨不耐脏，需定期打蜡；           | 客厅、卧室的地砖  |
| 哑光砖  | 有磨砂质感，防滑能力最好，相对抛光砖更避免光污染，但最容易脏             |           |
| 柔光砖  | 介于亮光砖与哑光砖之间，适合无主灯                          | 厨房的地砖、墙砖  |
| 柔抛砖  | 抛釉砖的一种，釉面经过柔抛成柔和的光度                        |           | 
| 抛釉砖  | 对通体砖上釉而成，防滑、耐脏不耐磨。像天鹅绒、金丝绒这一些，实质都是用不同釉的抛釉砖 | 厨房的地砖、墙砖  |
| 仿古砖  | 釉面瓷砖的一种                                    |           |
| 玻化砖  | 无釉瓷砖的一种，吸水率低于0.5%的瓷砖，且更耐磨，价格较贵             | 厨房、卫生间的地砖 |
| 马赛克  | 耐酸耐碱耐磨抗压，铺贴费用较贵                            | 局部造型、实验室  |

#### 如何看瓷砖属性

* **吸水率**：越低越防滑，通常要求不超过0.5%。（可以滴水上去，看容不容易留下水迹）
* **平整度**
* **耐火等级**
* **表面莫氏硬度**：表示物体的抗外界刮、磨、划的能力。瓷砖的一般在3~9之间。
* **耐磨性**：：国家标准分T1~T5 5个等级。T1最耐磨；T3用于日常家用；T4一般用于卫生间
* **破坏强度**
* **致密属性**
* **防污能力**：用油性笔测试砖的防污能力，如果轻轻擦拭就能去除污渍，说明好，反之则差
* **光泽度**：指瓷砖表面对于照入光线的折射能力。哑光砖——15°以下、柔光砖——25°—55°、55°以上——亮光砖（非统一标准）

## 各阶段的注意事项

### 墙体拆除注意事项

1. 确认好哪一块是承重墙，承重墙不能砸
2. 砸墙前地漏封胶带，避免碎石掉入
3. **围蔽**。阳台挂上保护网，避免灰尘和建筑垃圾往下掉
4. 关水电气阀门
5. 在要砸的墙体两面做**双面弹线**和**双面切割**
6. **切**。先用电锤沿线边缘切掉一部分，再用大锤砸
7. **清运费**。注意是否包含清运建筑垃圾的费用。
8. **回收**。拆下来的门窗、护栏、铝合金门、玻璃等，可以联系回收师傅卖点钱。

### 水电改造&验收注意事项

1. **电线选材**。选有3C认证的BV线，不选BVR多股软线
2. **电线规格**。进户线用BV10；中央空调、热水器用BV6；大功率电器用BV4；低功率电器用BV2.5；灯具、开关用BV1.5
3. **水管**。选PPR管不选PVC。
4. **网线**。六类或者超六类的非屏蔽网线。五类线最高网速：100Mbps、超五类：1000Mbps、六类：1Gbps、超六类：10Gbps
5. **预留点位**。要提前确认好灯具、电器、监控、洗衣机的布局，并且预留水电的点位。
6. **单独走线**。冰箱、弱电箱、监控要单独走一路线，做到出门断电不断网。
7. **全屋智能**。如果要做全屋智能，需要在开关处预留零线；如果要做mesh组网，或者全屋wifi覆盖，需要在每个房间预留网口。
8. **水电封槽**。水电线路安装完毕后，需要用水泥砂浆对线槽进行回填。
9. **线路布局**。点对点走线性价比最高，换线容易。切记留好布线的图纸，为日后维修做准备。
10. **强弱电交叉**。如果强弱电走线交叉了，需要在交叉处**包锡纸**，并且强电在上，弱电在下
11. 厨房水电建议走顶。
12. 电视墙记得提前预埋**50管**。
13. 洗衣机、扫地机器人要提前确认型号，按照要求预留水电位。
14. 浴室柜水管用**墙排**，不用**地排**。美观且不占用空间。
15. **打压测试**。水压0.8兆帕，稳压30~60分钟即合格。需要开质保单。


### （瓦工）瓷砖铺贴&验收注意事项


1. **瓷砖排版设计**。尽量把小条砖藏到柜子和家具下。
2. **瓷砖验收**。检查瓷砖等级、色号、3C认证
3. **铺贴时间**。厨房、洗手间要做完防水7天~10天左右后，才可进行铺贴。需要确认防水干了无漏水。
4. 墙上阳角要做**海棠角**，不做阳角条
5. 地砖通铺要**对缝**
6. 卫生间要**墙砖压地砖**，否则容易导致渗水
7. 卫生间要**墙地砖对缝**
8. 卫生间刷完防水之后，需要做**拉毛处理**，这样墙砖粘合更牢固
9. 地漏要做**回形地漏**
10. 阳台外铺砖要向外做点坡度。

### 卫生间施工注意事项

1. 基层清理，包括要铲掉开发商自带的防水
2. 用水泥找平，需要做点坡度，让水可以自然留到地漏
3. 要做**二次排水**的支管
4. 排污管、地漏、马桶管等地方要刷堵漏王，避免后期漏水
5. **刷防水**。墙面刷刚性防水，地面刷柔性防水；横竖刷一遍，要等第一遍干了再刷第二遍；淋浴区刷1.8米，洗手台刷0.6米，其他不低于0.3米
6. **闭水试验**。防水材料干透后，要做48小时闭水试验，注水要到20厘米。然后到楼下检查有没有漏水。
7. **材料回填**。防水搞定后，需要用材料回填。可选择陶粒回填、发泡水泥回填等
8. **再刷防水**。回填完需要再刷一遍防水。
9. 下水管道包隔音材料、用龙骨或轻体砖封起来
