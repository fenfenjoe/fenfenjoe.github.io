# 流行歌唱训练

- 控制气息，避免早早断气：**发si，数时长** 
- 腹式呼吸：**说话漏气的感觉**
- 胸式呼吸：**中低音用此方法**
- 换气练习：**狗喘气**

## 练习

【腹式呼吸】
- 一手放在胸口，一手放在肚脐下方小腹的地方
- 吸气，用鼻子或微张的嘴，想象气息直接吸到腹部。感受小腹的手被往外推，胸口的手则尽量不动。
- 呼气，用嘴巴缓慢、均匀、有控制地呼出（发出轻微的 "Sssss" 或 "Fffff" 声）。

【气息练习】
- **1.慢呼慢吸**
  - 缓慢吸气4S
  - 屏住呼吸2S
  - 缓慢、均匀呼吸8S，发"Ssss"音
- **2.快呼慢吸**
  - 快速无声吸气0.5~1S
  - 缓慢、均匀呼吸8S，发"Ssss"音
- **3.慢呼快吸**
- **4.快呼快吸（狗喘气）**

【气息支撑练习】
- 1.吸气，先找到横膈膜，然后吸气时想象横膈膜用力
- 2.呼气，保持两边肋骨往外撑，同时通过腹部收缩来啊把气呼出去

【音阶练习】
- 用 "Ah", "Ma", "La" 等音节，配合简单的音阶（如五度、八度音阶）或爬音练习

## 如何作出各种共鸣

整个过程总结：
1. 身体放松
2. 能正确做深长的腹式呼吸
3. 能在支撑下发声
4. 能将声音引导至共鸣区
5. 用滑音和音阶练习平滑连接它们

### 胸腔共鸣

【目标】  
胸口有震动感

【练习方法】  
- 自然叹气法
  1. 用腹式呼吸深呼吸
  2. 长叹气，发“唉”
- 低音哼鸣法
  1. 闭上嘴，发“M”
  2. 从低音缓慢滑向更低音，再滑回来，保持胸口震动感

【错误方法】  
不要压低喉结。这会导致声音僵硬、伤嗓。真正的胸腔共鸣是通畅、放松的。  

### 口腔/咽腔共鸣

【目标】  
保持软腭抬起

【练习方法】  
- 打哈欠法
  1. 深深打一个哈欠，并在喉咙最打开时发一个“哈”音
  2. 保持哈欠时喉咙后部那个“开阔”的感觉
- 抬软腭练习
  1. 发“哼”，结束时停住，感受软腭抬起
  2. 保持住软腭状态，发清晰的“啊”
- 元音聚焦练习
  1. 发“咦”或者“耶”这类闭口音、前元音
  2. 感觉声音像一束激光，集中地打在上门牙后方或硬腭上，产生一种明亮的“嗡嗡”振动感。这能有效提高声音的清晰度和投射力。

【错误方法】  
头腔共鸣 ≠ 鼻音。捏住鼻子，如果是正确的头腔共鸣，音色只会微变；如果是鼻音，声音会立刻被堵住。

### 头腔共鸣

【目标】  
找到高位置、集中、有金属质感的声音，感觉振动在眉心以上。

【练习方法】  
- 哼鸣练习
  1. 用“嗯”(M)音哼鸣，从较高但自己舒服的音区开始
  2. 感受哼鸣的震动点并逐步从鼻腔向上移动到鼻梁、眉心。
  3. 嘴唇轻轻闭合，牙齿微开。如果感觉鼻子发痒或眉心发麻，即为正确
- 鼻音引导练习
  1. 使用带鼻音的音节，如 “咪（Mi）”、“呐（Na）”、“应（Ing）”。
  2. 先带点鼻音哼出，然后慢慢打开嘴，但努力保持声音的位置和感觉还在鼻梁/眉心，不让它掉回嘴里。

### 常见错误
1. **胸口提气**：唱歌时肩膀紧张→解决：多练腹式呼吸，想象“气息沉到小腹”
2. **挤嗓子唱高音**：喉咙发紧→解决：用头腔共鸣，想象“声音往头顶送”
3. **低音靠喊**：声音干涩→解决：用胸腔共鸣，放松喉咙
4. **共鸣断层**：声音忽高忽低→解决：多练滑音，感受共鸣区的过渡
