英语学习笔记

日常

  • estimate vt. 预估、预计 n. 估算值

    • estimate...to be...(预估xx的数量约为...)
    • It's estimated that...(预计...)

  • estimated adj. 预估的、预计的

    • estimated time(预估时间)、estimated cost(预估成本)

  • workload n. 工作量

  • deliver vt. 交付

  • requirement n. 需求

  • 我觉得有可能是因为...

    • It might be...
    • I suspect that...
    • The reason may be that

Git相关

git文档（可以学习相关术语）：https://git-scm.com/

常用短语：

  • 拉取最新代码到本地： Pull the latest code to local
  • 创建一个新分支去完成功能： Create a new branch from origin/master
  • 提交代码：Commit the changes
  • 发起Pull Request(PR)并等待代码审查(CR)：Create a Pull Request (PR) and wait for Code Review (CR).
  • 撤销提交：Revert the commits

连接副词

词/短语含义/作用例句
因果关系
therefore因此The component is decoupled; therefore, it is easier to test.
consequently因此，所以The service failed; consequently, the whole system crashed.
as a result结果He forgot to close the connection; as a result, a memory leak occurred.
thus因此，从而The code is refactored, thus reducing technical debt.
hence因此，由此The API is deprecated; hence, we should migrate to the new one.
递进与补充
furthermore此外，而且The solution is scalable; furthermore, it is secure.
moreover此外，更重要的是The design is clean; moreover, it meets all requirements.
in addition另外In addition, we need to update the documentation.
additionally此外Additionally, the response time improved by 20%.
besides此外，而且The approach is simple; besides, it requires no extra dependencies.
转折与对比
however然而The system is fast; however, it consumes too much memory.
nevertheless尽管如此The task was challenging; nevertheless, we completed it on time.
nonetheless尽管如此It's an old library; nonetheless, it's still widely used.
on the other hand另一方面This solution is cheap; on the other hand, it lacks flexibility.
in contrast相比之下Java uses static typing; in contrast, Python is dynamically typed.
conversely相反地Adding more threads may improve performance; conversely, it could increase contention.
顺序与时间
first / firstly首先First, we need to define the interface.
second / secondly其次Secondly, implement the business logic.
finally最后Finally, deploy the application.
meanwhile与此同时The frontend team builds the UI; meanwhile, the backend team sets up the API.
subsequently随后The bug was fixed; subsequently, the build passed.
举例与说明
for example例如Many frameworks support DI; for example, Spring and Guice.
for instance例如Consider a microservice architecture; for instance, order service and payment service.
such as比如 (通常后接名词)We use various tools such as Maven, Git, and Jenkins.
namely即，也就是There is one key benefit, namely, reduced coupling.
总结与结论
in conclusion总之In conclusion, dependency injection simplifies testing.
to sum up总结To sum up, IOC containers manage object lifecycles.
overall总体上Overall, the migration was successful.
in summary概括来说In summary, the new architecture improves maintainability.
强调与重述
in other words换句话说The bean is singleton; in other words, only one instance exists.
that is也就是说The method is idempotent; that is, calling it multiple times has the same effect.
indeed确实Indeed, the performance test confirmed our hypothesis.
条件与让步
otherwise否则We must handle the exception; otherwise, the application may crash.
in that case那样的话If the database is down, in that case, we fall back to cache.
类比与比较
similarly同样地Microservices scale independently; similarly, each can be deployed separately.
likewise同样The service requires authentication; likewise, all endpoints should be secured.
表达目的
to this end为此We need better monitoring; to this end, we will implement Prometheus.
for this purpose为此A dedicated thread pool was created; for this purpose, we used Executors.
